||||| (Open Local html on Kinkle Fire

1. Rename xxx.html to xxx.txt [ Why ? If you upload .html to kindle, it insists on retaining the full path on desktop pc where file originated. Since that path does not exist on Kindle, file will not open. ] 2. Using python3 -m http.server 9000 (or any other web server) open http://192.168.1.122:9000 on kindle silk browser. 3. on kindle, right click on xxxx.txt and download. 4. on kindle open "Filer" app. Long hold to highlight xxxx.txt and in top right choose move. Then in top left dropdown choose Documents folder. Click tiny blue Move button in bottom right. 5. Using Top right dropdown, select Documents folder. Long hold xxxx.txt and select Open in HTML Viewer. 6. Download and move to Documents and open in HTML Viewer.
